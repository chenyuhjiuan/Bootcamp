Clone an Existing Repository



The complete source code for git is stored on GitHub at https://github.com/git/git (Links to an external site.). Y ou can create a complete clone of this project on your computer in about 30 seconds. Wouldn't that be cool? This is cool because you can find projects you like and clone them to either explore their code and learn more about coding or to make changes to an existing project. You'd need to contact the project owner to find out how they might want you to submit change requests, if any, so we won't go there. Just know it's possible.

To clone the git project on your computer:

Using the Git shell, create a directory called git Change to that directory Type git clone https://github.com/git/git (Links to an external site.) Wait just a minute or two for all the files to finish downloading.

You now have the git source code on your computer.
